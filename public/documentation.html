Objective

To build the server-side component of a “movies” web application.
The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their- personal information, and create a list of their favorite movies.
The complete tech stack used is MERN (MongoDB, Express, React, and Node.js) stack.

Stack and dependencies

Stack: MongoDB, Express, React, and Node.js
Dependencies: express, morgan, body-parser

Essential Features

Request Query parameter URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies /movies GET None JSON object holding data about all the movies
Return data about a single movie by title to the user /movies/:title /movies/[Title] GET None JSON object holding data about a single movie by title.
E.g.
{'Title': '12 Angry Man',
'Description': 'The film tells the story of a jury of 12 men',
'Genre': 'Drama',
'Director': 'Sidney Lumet',
'Image URL': 'shorturl.at/kuF57',
'Featured': 'True' }
Return data about a genre by name/title /genre/:name /genre/[Name] GET None JSON object holding data about all genres
E.g.
{'Genre': 'Drama',
'Description': 'Drama is a category of narrative fiction (or semi-fiction) intended to be more serious than humorous in tone.'}
Return data about a director by name /director/:name /director/[Name] GET None JSON object holding data about all directors
E.g.
{'Name': 'Martin Scorsese',
'Birthday': '1942-11-17',
'Nationality': 'american, italian'}
Allow new users to register /users /users POST A JSON object holding data about the user to register
E.g.
{'Name': 'John Doe',
'Birthday': '1995-12-05',
'Email': 'john.doe@gmail.com'}
 A JSON object holding data about the user that was registered
E.g.
{'ID': '123456',
'Name': 'John Doe',
'Birthday': '1995-12-05',
'Email': 'john.doe@gmail.com'}
Allow users to update their user info /users/:username /users/[Username] PUT JSON object holding data about the users update
E.g.
{'Name': 'Jane Doe'}
 A text message indicating the info that was updated
E.g.
{'Name': 'Jane Doe'}
Allow users to add a movie to their list of favorites /users/:username/Movies/:movieID /users/[Username]/Movies/[MovieID] POST A JSON Object holding data about the movies to add to the favorites list
E.g.
{'FavMovie': 'Inception'}
 A JSON Object holding data about the movies that was added to the favorites list
E.g.
{'FavMovie': 'Inception'}
Allow users to remove a movie from their list of favorites /users/:username/Movies/:title /users/[Username]/Movies/[Title] DELETE None A text message indicating whether the movies was successfully removed from favorites
Allow existing users to deregister /users/:username /users/[Username] DELETE None A text message indicating whether the account has been deleted
Allow users to see which actors star in which movies /actors /actors GET None JSON object holding data about all actors
Allow users to view information about different actors /actors/:name /actors/[Name] GET None JSON object holding details about the chosen actor
Allow users to view more information about different movies /movies/:title /movies/[Title] GET None JSON object holding details about the chosen movie (release date and movie rating)
Allow users to create a “To Watch” list in addition to their “Favorite Movies” list /users/:username/favorites/watchlist /users/[Username]/favorites/watchlist POST JSON object holding "To Watch" movie data JSON object holding data that was added to the "To Watch" list
Allow users to delete a movie from the “To Watch” list /users/:username/favorites/watchlist/:title /users/[Username]/favorites/watchlist/[Title] DELETE JSON object holding "To Watch" movie data JSON object holding data that was added to the "To Watch" list
